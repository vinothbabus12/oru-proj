# BusBook — Login Page

Static demo login page for the online bus ticket booking project.

How to use

- Open `login.html` in your browser (double-click or serve with a static server).
- Enter any identifier and use password `password` to see a successful demo login.

Run the included demo server (optional)

1. Open a terminal in the `template` folder.
2. Install dependencies and start the server:

```bash
npm install
npm start
```

3. Open http://localhost:3000/index.html

4. Click "Open Login" to open the login page.

5. After login success, you will be redirected to the bus search page where you can choose departure and arrival cities.

To integrate with your backend: attach an API POST to the form submit handler in `script.js`.
